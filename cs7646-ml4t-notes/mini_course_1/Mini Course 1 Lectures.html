def compute_daily_returns(df):
    """Compute and return the daily return values."""
    # TODO: Your code here
    # Note: Returned DataFrame must have the same number of rows
    dr = df.copy()
    dr[1:] = (df[1:] / df[:-1].values) - 1       # .values needed because pandas wants to match up the indeces, which                 #                                                   won’t work with 2 different date indeces
    dr.ix[0] = 0
    return dr

01-05 Incomplete Data

gaps/missing data even for stock data
not all stocks trade

Why Data Goes Missing

An ETF or Exchange-Traded Fund is a basket of equities allocated in such a way that the overall portfolio tracks the performance of a stock exchange index. ETFs can be bought and sold on the market like shares.
For example, SPY tracks the S&P 500 index (Standard & Poor's selection of 500 large publicly-traded companies).
You will learn more about ETFs and other funds in the second mini-course.

If missing data, fill forward, then backward the last known price
pandas.DataFrame.fillna()
method = ‘pad’ or method = ‘ffill’ fills forward from last known
method = ‘backfill’ or method = ‘bfill’ fills backward from next known

- position_vals = allocation * start_value
- portfolio_val = position_vals.sum(axis = 1)

cum_return = (portfolio_val[-1]/portfolio_val[0]) - 1

Sharpe ratio (SR) - risk-adjusted return (William Sharpe)
 - all else equal, lower risk is better and higher return is better
 - also considers risk-free rate of return, i.e., savings account interest rate
 - SR = Expected ( portfolio return - risk free return ) / std of ( portfolio return - risk free return )
 - in python = mean ( daily_rets - daily_riskfree ) / std ( daily_rets - daily_riskfree )

LIBOR (London Interbank Rate )
3-month T-bill
0%

daily risk free rate = 252nd-root ( 1 + annual/term rate ) - 1

if daily risk free rate doesn’t change, it can be dropped from std calc

sharpe ratio can vary widely depending on how frequently it is sampled (daily, weekly, monthly).

SR_annualized = K * SR, k = sqrt( samples per year )


01-08 Optimizers: Building a parameterized model

Optimizers - algorithm that can:
  • find minimum values of functions
  • build a parameterized model based on data
  • refine allocations to stocks in portfolios

3 key steps
- define a function to minimize - f(x) = x2 + 0.5  - python will run many times to find best answer
- provide an initial guess
- call the optimizer

import pandas as pd
import matplotlib.pyplot as plt
import numpy as np
import scipy.optimize as spo

def f(X):
    Y = X**2 + X**3 + 5
    print "X = {}, Y = {}".format(X, Y)
    return Y

def test_run():
    Xguess = 2.0
    min_result = spo.minimize(f, Xguess, method = 'SLSQP', options = {'disp': True})
    print "minima found at:"
    print "X = {}, Y = {}".format(min_result.x, min_result.fun)

if __name__ == "__main__":
    test_run()
